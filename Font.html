MEDIUM

DESCRIPTION

In this project i have used lightbox to do the font tracing. This work has gotten me to practice more on my hand skill to improve in font designs. Fonts can be used in many art projects such as greeting cards, posters, magazine and many more. In order to make it look perfect, i have used ruler as my guide line so that the shape of the fonts will look straight and neat.
